# Exely Telegram Bot

Telegram bot MVP for apartment booking with future Exely PMS integration.

## Railway variables

Add these variables in Railway:

- BOT_TOKEN
- MANAGER_CHAT_ID optional
- EXELY_API_BASE_URL optional for now
- EXELY_API_KEY optional for now
- EXELY_PROPERTY_ID optional for now

## Start command

python -m app.main
